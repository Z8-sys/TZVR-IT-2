# TZVR-IT-2

## О проекте

Данное простое консольное приложение на C++ реализует классы геометрических объектов (круг, треугольник, прямоугольник), наследуемых от абстрактного класса.

## Реализованные классы

- `GeometricShape` - Абстрактный класс для геометрических фигур.
- `Circle` - Класс, реализующий круг.
- `Triangle` - Класс, реализующий треугольник.
- `Rectangle` - Класс, реализующий прямоугольник.

## Особенности

- В каждом классе реализован метод расчета площади.
- Реализована сортировка массива геометрических фигур по площади.
- Добавлены автотесты с разными тестовыми данными для проверки граничных условий.

## Запуск проекта

### Через Visual Studio

1. Откройте Visual Studio.
2. Нажмите **"Клонировать репозиторий"**.
3. Вставьте ссылку на репозиторий GitHub.
4. Дождитесь загрузки проекта.
5. Нажмите **"Запуск"** (или F5) для старта приложения.

### Через командную строку (MSVC)

1. Откройте **x64 Native Tools Command Prompt for VS**.
2. Перейдите в папку с проектом.
3. Скомпилируйте проект командой:
   ```sh
   cl /EHsc /std:c++17 /Fe:geometry_app.exe TZVR-IT.cpp Circle.cpp Rectangle.cpp Triangle.cpp
   ```
4. Запустите приложение:
   ```sh
   geometry_app.exe
   ```

## Основные файлы

- `TZVR-IT.cpp` - Главный файл проекта.
- `GeomtricShape.h` - Реализация классов геометрических фигур.
